# Лабораторна робота №1: Мобільний додаток на React Native
**Виконала:** Литвинчук Ольга, група ІПЗ-24-3

## Опис проєкту
Мобільний додаток, розроблений за допомогою Expo та React Native. Реалізовано три основні екрани з навігацією:
* **Головна:** стрічка новин університету.
* **Фотогалерея:** сітка із зображеннями.
* **Профіль:** форма реєстрації користувача.

## Як запустити
1. Встановити залежності: `npm install`
2. Запустити проєкт: `npx expo start`
3. Відсканувати QR-код через додаток Expo Go.
